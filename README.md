# 💰 FinWise – Personal Finance Management Dashboard

A modern finance management dashboard built using **HTML, CSS, and JavaScript**, featuring a responsive banking-style interface with secure authentication using **Amazon Cognito** and designed for deployment on **AWS Amplify**.

---

## 🚀 Live Demo


https://main.d3f9thmqek52uo.amplifyapp.com

## ✨ Features

- 🔐 Secure user authentication with Amazon Cognito
- 👤 User Sign Up, Sign In & Email Verification
- 📊 Interactive Finance Dashboard
- 💳 Account Overview
- 💰 Income & Expense Tracking
- 📈 Cash Flow Analytics
- 🥧 Spending Category Chart
- 📋 Transaction Management
- 📂 Transaction Folder Organization
- 🎯 Budget Tracking
- 📑 Financial Reports
- ⚙️ User Settings
- 🌙 Dark Theme
- ☀️ Light Theme
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

### Frontend
React / Next.js
Tailwind CSS
Chart.js or Recharts

### AWS Services
AWS Lambda          -> backend API functions
Amazon API Gateway -> REST API routes
Amazon DynamoDB    -> database
Amazon Cognito     -> authentication
Amazon S3          -> file storage if needed
AWS Amplify        -> frontend hosting + AWS integration
AWS SDK            -> access AWS services from backend

### Development Tools
- Git
- GitHub
- VS Code

The application is deployed using **AWS Amplify**.

Deployment process:

1. Push project to GitHub
2. Connect repository to AWS Amplify
3. Deploy automatically
4. Access via Amplify URL
